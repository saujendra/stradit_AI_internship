## What is a Function?
A function is a named, reusable block of code that performs a specific task. Instead of
repeating code, you write it once inside a function and 'call' it whenever needed.
• Avoid repetition — Write once, use many times (DRY: Don't Repeat Yourself)
• Organise code — Break large programs into smaller, manageable pieces
• Make code readable — A well-named function explains what it does
• Enable testing — Test each function independently

1. Single argument
Function takes one value.


2. Multiple arguments
Function takes more than one value.

3. Positional arguments

4. Default arguments

Function already has a default value.

5. Keyword arguments

6. Variable length arguments (*args)
Function can take many values at once.
